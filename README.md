# Price Tracker

A smart shopping assistant that tracks items you want to buy, alerts you when prices drop, and finds similar alternatives.

## Features

- **Wishlist Management** - Keep track of items you want to purchase
- **Price Alerts** - Get notified when prices drop below your target
- **Similar Item Discovery** - AI-powered recommendations for comparable products
- **Price History** - Track price trends over time

## Tech Stack

- **Backend**: Python (FastAPI)
- **Database**: PostgreSQL
- **Frontend**: React + TypeScript
- **AI/ML**: OpenAI API for similarity matching
- **Price Scraping**: Beautiful Soup / Playwright
- **Notifications**: Email / Push notifications
